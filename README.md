# StockFlow Low-Stock Alerts API

This project demonstrates a backend API that generates low-stock alerts
for a B2B inventory management system.

## Features
- Multi-warehouse inventory handling
- Product-specific low-stock thresholds
- Recent sales-based alerting
- Supplier information for reordering

## Tech Stack
- Python
- Flask
- SQLAlchemy
- SQLite

## How to Run
1. Create virtual environment
2. Install dependencies
3. Seed database
4. Run Flask server

See detailed instructions in the code comments.
